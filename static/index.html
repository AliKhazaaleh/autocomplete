Autocomplete API

Welcome to the Autocomplete API. This service provides intelligent autocomplete suggestions for various categories.

Available Endpoints

Countries

GET /countries?q=united&limit=10

Get country suggestions (200+ countries)

Cities

GET /cities?q=new&limit=20

Get city suggestions (74000+ cities)

Universities

GET /universities?q=university&limit=5

Get university suggestions (0+ universities)

Companies

GET /companies?q=tech&limit=10

Get company suggestions (0+ companies)

Job Titles

GET /job-titles?q=engineer&limit=10

Get job title suggestions (850+ job titles)

Skills

GET /skills?q=java&limit=10

Get skill suggestions (87000+ skills)

Query Parameters

  • q: Search query string (optional)
  • limit: Maximum number of results (optional, default: 20, max: 20)
